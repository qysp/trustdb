Update all documents in the collection that match the query or pass the custom filter function.

Parameters:
Name Type Description
filter object | function

query object or filter function which will be applied on each document
updater object | function

object or update function which will be applied on each filtered document
Source:
  • collection.js, line 285
Fires:
  • Collection#event:update after successfully updating the documents
Returns:

a Promise with the updated documents

Type
Promise.<(Array.<object>|Error)>
Example
// Update all documents where the value of the property
// price is greater than or equal to 50 and set it to 49.99.
await collection.update(
  doc => doc.price >= 50,
  { price: 49.99 }
);
// or
await collection.update(
  { price: { greaterThanOrEqual: 50 } },
  doc => doc.price = 49.99
);
