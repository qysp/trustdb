connect(filepath, settingsopt) → {Promise.<(void|NodeJS.ErrnoException)>}

Create a connection to a new database. If the file in the filepath already exists, it will be loaded and the collections (and optionally the settings) will be applied to this instance, unless the setting overwriteExisting is set to true.
